# Visual Brief

## Direction

No stock dependency. This episode should look like an internal software workflow that became sentient enough to assign itself tickets but not sentient enough to own the merge.

Use a mix of:

- Kinetic typography for conceptual pivots.
- Code mockups for agent/tooling moments.
- JIRA/Slack/calendar mockups for procedural comedy.
- Cards only for opening and final button.

## Scene Notes

- `s01`: Static DHH setup card.
- `s02`: Kinetic "6" to make the public shift feel immediate.
- `s03`: Code mockup showing the difference between typing and agent handoff.
- `s04`: Kinetic token/autocomplete contrast.
- `s05`: JIRA ticket assigned to Codex, reporter DHH.
- `s06`: Slack handoff from Codex to human reviewer.
- `s07`: Kinetic thesis card: review is the job.
- `s08`: Calendar invite for the merge committee of one.
- `s09`: Diff mockup with the taste dependency.
- `s10`: LGTM button.

## Avoid

- Missing b-roll fallbacks.
- Generic office clips.
- Robot imagery.
- Vendor-demo language.
